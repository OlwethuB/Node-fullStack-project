API Descriptions

HTTP Method Path Description
Home
Get /(home page) This directs us to the the 'front' page, the very first thing we see when we open or run the app.
Users
Get /users Display a list of users
Get /user/:id Display a single user
Post /user To register a new user
Patch/Put /users/:id To modify a user’s record OR Update user’s record
Delete /user/:id To delete a single user
Products
Get /products Display a list of products
Get /products/:id Display a single product
Post /products To add a new product
Put/Patch /product/:id Modify or replace a product’s record
Delete /product/:id Delete a product
